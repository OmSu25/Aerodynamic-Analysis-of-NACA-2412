# ✈️ Aerodynamic Analysis of NACA 2412 Airfoil

This project is an interactive aerodynamic simulator developed in Python to analyze NACA 4-digit airfoils.

It demonstrates lift generation and pressure distribution using Thin Airfoil Theory along with real-time flow visualization.

---

## 🚀 Features

- NACA 4-digit airfoil generation  
- Lift coefficient (Cl) calculation using Thin Airfoil Theory  
- Pressure coefficient (Cp) visualization  
- Streamline flow visualization  
- Interactive GUI with angle of attack control
